\# 🤖 EJCP: Embodied Joint Control Platform

\*\*分布式具身智能柔性关节控制平台 V1.0\*\*



\## 🌟 项目简介

EJCP 是一个为机器人关节、外骨骼、协作机械臂打造的\*\*“大小脑解耦”\*\*分布式驱动平台。

本项目通过 ESP32-S3 (决策大脑) 与 HW3511 一体化无刷电机 (运动小脑) 的软硬件协同，实现了极其平滑的 FOC 矢量控制、无传感柔顺阻抗、以及防炸机安全护盾。



\## ⚙️ 硬件基线 (Baseline)

\- \*\*大脑 (Upper)\*\*: ESP32-S3-DevKitC-1 (N16R8)

\- \*\*小脑 (Lower)\*\*: HW3511 2804 智能无刷模组 (STM32 + DRV8311)

\- \*\*物理链路\*\*: UART (9600 Baud, 8N1)

\- \*\*接线规范\*\*:

&#x20; - `电机 PORT2 蓝线` ➡️ `ESP32 GND` (必须共地)

&#x20; - `电机 PORT2 绿线 (TX)` ➡️ `ESP32 Pin 8 (RX)`

&#x20; - `电机 PORT2 黄线 (RX)` ➡️ `ESP32 Pin 9 (TX)`



\## 🚀 快速点火 (Quick Start)

1\. 将小脑固件（降频至 9600）烧录至 HW3511。

2\. 将 `Brain\_ESP32` 工程通过 PlatformIO 烧录至 ESP32。

3\. 接通电机 12V 动力电源。

4\. 等待 8 秒开机自检后，电机将自动注入安全阻抗参数（MAP/MAD/MVI/MLU），并进入自动化优雅折返跑模式。
