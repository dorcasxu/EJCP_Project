智能柔性关节-单关节执行器工业化实施标准书 V1.0
**（依据 EJCP 分布式柔性关节平台白皮书制定）**
**发布目标**：将桌面原型转化为“可规模化复制、防呆、安全”的工业级模组交付标准。

### 一、 核心架构定义（Tier 2 标准）
本产品禁止采用“大脑包揽一切”的单片机模式，必须严格执行**【分布式解耦架构】**：
1.  **执行层（小脑 / HW3511）**：作为黑盒交付。只负责高频（10kHz）的 FOC 电流环/位置环计算、传感器读取、底层硬件限幅（防烧毁）。
2.  **决策层（大脑 / ESP32-S3）**：作为网关与逻辑层交付。只负责轨迹下发、工况状态机切换（刚柔切换）、以及未来的“痛觉感知与碰撞反射逻辑”。
3.  **通信层**：UART 9600 波特率。稳定压倒一切，宁可降速，绝不允许出现丢包引起的失控暴冲。

### 二、 工业级开机部署标准流程 (SOP)
任何一台新组装的关节，上电流程必须严格遵循以下状态机（State Machine），绝不允许“抢跑”：
*   **【State 0: INIT】** 12V 通电。大脑强制静默 **8 秒**，等待小脑内部的磁场对齐与电流传感器零偏校准彻底完成。
*   **【State 1: CONFIG】** 大脑下发底层工况配置：`@3\n` (角度模式) ➡️ `MAP/MAD/MVI/MLU\n` (刚柔参数集)。
*   **【State 2: ENABLED】** 大脑下发 `ME1\n`，电机发声并锁死进入工作态。
*   **【State 3: RUNNING】** 大脑开始以低频（如 10Hz~50Hz）下发位置指令 `Mx.xx\n`。

### 三、 工业级安全与故障闭环（核心壁垒）
“柔性”只是表象，“安全”才是工业品的命脉。本平台实施三级保护策略：
1.  **L1 硬件级防烧（已实现）**：通过固化 `MLUx.x` (电压限幅)，即使发生死锁、被人死死卡住，电机也只能输出设定的最大功率，**从根本上杜绝了线圈过热烧毁的物理可能**。
2.  **L2 运动学防震（已实现）**：通过固化 `MVLx.x` (速度限幅) 和 `MVI` (积分清零)，彻底消灭了阶跃指令引起的起步暴冲（Jerk）和到位抽搐，实现真正的“优雅阻抗”。
3.  **L3 主动碰撞卸力（下一阶段开发重点！）**：大脑实时监听小脑反馈的误差/力矩数据，一旦超出安全阈值，大脑瞬间下发 `ME0\n`（软瘫痪卸力），实现“打人不疼、砸不坏机器”的植物神经反射。

### 四、 参数模板库（SKU 场景交付预设）
针对不同的客户场景，大脑开机时直接调用以下“黄金参数套餐”：
*   **📦 场景 A：人机协作/康复柔顺模式**
    *   参数集：`MAP1.0` (极软), `MAD0.1` (微阻尼), `MLU2.0` (低压安全), `MVI0` (无记忆)。
    *   特征：像海绵一样，随手可掰动，撞人毫无痛感。
*   **📦 场景 B：重载精准指向模式**
    *   参数集：`MAP4.0` (中等偏硬), `MAD0.1` (防震荡), `MLU5.0` (充足动力), `MVI5.0` (开启积分)。
    *   特征：指哪打哪，能克服静摩擦力，死磕到底，彻底消除稳态误差（转不到位现象）。
